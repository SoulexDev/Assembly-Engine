The Assembly Engine is the in-house engine being dveloped by The Dev Assembly. It is a lightweight, code based game engine. It is designed for performance, ease of use, and freedom, without the hassle of doing absolutely everything yourself.

#Plans
Currently, I am looking to implement these features -
* Massive shadows
* Penumbra shadows
* Voxelized shadows
* Various lighting methods
* Volumetric rendering

#Tools
Eventually, I will create tools to assist with building games in the engine
* Shader editor
* Level editor(scene editor)
* Modeling tool(various modeling modes)
